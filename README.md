## projeto-pytube
- projeto para download de músicas e videos do youtube
- uso da bliblioteca pytube para download do link 
- e uso da bliblioteca tkinter para tela 
- ainda em fase de testes

## como usar 
- abra a pasta "dis/" e execute o arquivo "janela.py"
- copie um link e clique em "download"
- para baixar mais de um link clique em "adicionar"
- caso queria converter todos para mp3, clique em "converter"
